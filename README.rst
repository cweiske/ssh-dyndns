**********
ssh-dyndns
**********
Update DNS records by simply SSH'ing into a server.

Multiple implementations available:

- djbdns/dbndns ``tinydns``: See the ``djbdns/`` directory
- NLnet Labs Name Server Daemon NSD: See the ``nsd/`` directory
